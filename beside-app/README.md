# Beside — Someone's here.

一個溫暖的陪伴應用程式，讓焦慮的人們能夠找到彼此的支持。

## 🌟 功能特色

- **焦慮信號發送**：當你感到焦慮時，可以發送信號讓附近的人知道你需要陪伴
- **附近列表**：瀏覽附近正在尋求陪伴的人
- **溫暖回應**：簡單的「我陪你」回應，傳遞溫暖
- **即時互動**：看到有多少人回應你的信號
- **隱私保護**：無法聊天，只有簡單的陪伴確認

## 🎨 設計理念

- **溫暖色調**：使用粉紅色系營造溫暖氛圍
- **簡潔界面**：專注於核心功能，避免複雜操作
- **情感設計**：每個元素都傳遞溫暖和支持的感覺

## 🚀 技術棧

- **框架**：Next.js 15 + React 18
- **樣式**：Tailwind CSS + 自定義 CSS
- **動畫**：Framer Motion
- **圖標**：Lucide React
- **狀態管理**：React Context
- **類型安全**：TypeScript

## 📱 頁面結構

1. **歡迎頁**：應用介紹和開始使用
2. **註冊頁**：創建用戶資料
3. **登入頁**：手機號碼驗證登入
4. **主頁**：發送焦慮信號或查看回應
5. **附近頁**：瀏覽附近需要陪伴的人
6. **收到頁**：查看收到的回應
7. **設置頁**：個人資料和隱私設置

## 🎯 核心交互流程

1. **用戶A發送焦慮信號**：點擊主頁按鈕 → 信號發送到附近列表
2. **用戶B看到並回應**：瀏覽附近列表 → 選擇一個 → 點擊「我陪你」
3. **用戶A收到回應**：主頁顯示回應數量和列表
4. **信號管理**：可隨時取消信號

## 🎨 設計規範

### 顏色系統
- **主色調**：#FF6B6B (溫暖紅)
- **輔助色**：#4ECDC4 (青綠)
- **背景色**：#FFF5F5, #FFE5E5
- **文字色**：#333 (主), #666 (次), #999 (輔)

### 間距系統
- **主要間距**：30px, 20px, 15px
- **圓角**：按鈕30px, 卡片20px
- **陰影**：0 10px 30px rgba(0,0,0,0.08)

### 字體系統
- **標題**：24px, 粗體
- **副標題**：18-20px, 中粗
- **正文**：14-16px, 常規
- **輔助文字**：12-13px, 常規
